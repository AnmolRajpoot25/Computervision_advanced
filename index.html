AI Trainer

Pose Estimation Repetition Counter Angle-based Feedback

The AI Trainer script uses MediaPipe Pose to extract 2D skeletal landmarks and compute joint angles in real time. It tracks motion patterns, counts repetitions for exercises like squats or bicep curls, and provides posture-aware feedback based on angular thresholds.

Input: Webcam / video stream Stack: Python, OpenCV, MediaPipe

AI Painter

Hand Tracking Gesture Drawing Virtual Canvas

The AI Painter module turns your hand into a virtual brush using MediaPipe Hands. It tracks fingertip coordinates, maps them to canvas space, and renders strokes with configurable brush thickness and eraser modes to create a touchless drawing interface on top of the video feed.

Input: Webcam + hand gestures Stack: Python, OpenCV, MediaPipe
